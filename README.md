# Telegram Role Shop Bot 🛒

Бот для покупки ролей в Telegram чате с системой монет и промокодов.

## 🎭 Роли
- VIP - 12.000 монет
- Pro - 15.000 монет  
- Phoenix - 25.000 монет
- Dragon - 40.000 монет

## 💰 Заработок
- За сообщение в чате: +1 монета
- За приглашение друга: +100 монет
- Промокоды от админа

## 🛠 Команды для всех
- `/start` - регистрация
- `/top` - топ игроков
- `/invite` - реферальная ссылка
- `/promo` - информация о промокодах
- `/use КОД` - активировать промокод

## 👑 Админ-команды
- `/addcoins ID СУММА` - выдать монеты
- `/removecoins ID СУММА` - забрать монеты
- `/giveall СУММА` - выдать всем
- `/createpromo КОД МОНЕТЫ ИСПОЛЬЗОВАНИЯ [ДНИ]` - создать промокод
- `/listpromo` - список промокодов
- `/stats` - статистика
- `/users_1` - список пользователей
